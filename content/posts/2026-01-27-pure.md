[<cite>pure</cite>](https://github.com/sindresorhus/pure) by Sindre Sorhus is a prompt enhancement for ZSH I like a lot.  

<figure class="max-w-sm">
<img src="/img/code/pure.png" >
<figcaption>My pure prompt. I like to see my current folder and the Git status in a dedicated line, followed by a new line for inputting my commands.</figcaption>
</figure>

My settings in the `.zshrc` file:

```sh
# .zshrc 
# Pure prompt
fpath+=("$(brew --prefix)/share/zsh/site-functions")
autoload -U promptinit; promptinit
prompt pure
zstyle :prompt:pure:path color cyan
zstyle :prompt:pure:git:branch color yellow
```

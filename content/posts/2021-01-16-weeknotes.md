- David Seidman on _[how to add commas between a list of items dynamically with CSS](https://css-tricks.com/how-to-add-commas-between-a-list-of-items-dynamically-with-css/)._ I like the idea of using the sibling selector to add something between two siblings :-).
  ```css
  li ~ li::before {
    content: ", ";
  }
  ```

---
title: Freeze
tags: app
abstract: Make images from text
---
[<cite>Freeze</cite>](https://github.com/charmbracelet/freeze) can create images from text files and from shell command output. E.g.

```sh
freeze --execute "ls -ls" -o freeze-ls-ls.png
```

will create a file `freeze-ls-ls.png` that shows the result of the `ls -ls` command, which looks like:

<figure class="max-w-prose">
<img src="/img/code/freeze-ls-ls.png">
<figcaption>Output of <code>freeze --execute "ls -ls" -o freeze-ls-ls.png</code></figcaption>
</figure>

You can get an image of a text file, like a `go.mod` file, as well. Issue the command

```sh
freeze go.mod -o freeze-go-mod.png
```

and get the following image:

<figure class="max-w-prose">
<img src="/img/code/freeze-go-mod.png">
<figcaption>Output of <code>freeze go.mod -o freeze-go-mod.png</code></figcaption>
</figure>

Because Freeze is written in Go, you can use it on any operating system.
